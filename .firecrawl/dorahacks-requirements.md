# Judging Criteria by Prize

**An overview of each track is that all of the projects can be executed either via an AI agent running completely, or through manipulated AI or human-developed AI tools.**

# **Grand Champion**

The highest honor of this hackathon, awarded to the project that demonstrates excellence across technology, innovation, and ecosystem contribution — regardless of track.

### **Winning Path**

Projects must demonstrate outstanding performance across **all** dimensions. Open to all tracks.

### **Scoring**

| Dimension | Weight | Description |
| :-- | :-- | :-- |
| Technical Depth | 30% | AI × on-chain integration, architecture completeness, code quality |
| Innovation | 25% | Originality of solution, whether it proposes a new AI × Web3 paradigm |
| Mantle Ecosystem Contribution | 25% | Substantive use of Mantle network and long-term value to the ecosystem |
| Product Completeness | 20% | Runnable demo, user experience, scalability |

### **Requirements**

- Must be deployed on Mantle Network
- Submit open-source repo + runnable demo + project pitch
- Must be nominated from at least one track

# **By Tracks — First Prize**

# **Alpha & Data Track _(Exclusively Sponsored by Mirana Ventures)_**

Use AI to unlock the value of on-chain data — build analytics or trading tools that generate real, verifiable Alpha.

### **Two Winning Paths**

**A. \[Human - Driven\] Data & Analytics** — Build AI-powered on-chain data analysis, monitoring, or prediction tools

**B. \[AI - Driven\] Trading Strategy** — Build executable AI trading agents that generate verifiable on-chain Alpha

### **Scoring**

**General (60%):** Data source quality / AI analysis depth / technical completeness / sustainability

**Track-Specific (40%)**

- **Data & Analytics → Insight Value:** Uniqueness of findings + data visualization quality
- **Trading Strategy → Strategy Alpha:** Strategy complexity + verifiability (backtesting / live trading / on-chain records)

### **Encouraged Directions**

- Smart money tracking agent
- Mantle ecosystem protocol data dashboard
- AI-driven market sentiment analysis
- Automated arbitrage / market-making strategies

### **Requirements**

- Use Mantle on-chain data as a core data source
- Deploy on Mantle Network
- Submit open-source repo + demo + one-line pitch

### **Tell Us in Your Submission**

Which data sources does your project use? What role does AI play? How does it generate verifiable value on Mantle?

# **AI & RWA Track _(Exclusively Supported by Mantle Network)_**

Use AI to bridge real-world assets with the on-chain world — making RWA smarter and more accessible.

### **Two Winning Paths**

**A. \[Human Driven\] RWA Infrastructure** — Build AI-powered tools for RWA tokenization, pricing, or compliance

**B. \[AI Driven\] RWA Application** — Build end-user-facing AI × RWA products that lower the barrier to real-asset investing

### **Scoring**

**General (60%):** Depth of AI × RWA integration / technical completeness / Mantle integration / compliance awareness

**Track-Specific (40%)**

- **Infrastructure → Technical Feasibility:** Completeness of asset tokenization flow + innovation of technical approach
- **Application → Real-World Validity:** Clear asset category + well-defined target users + complete user experience

### **Encouraged Directions**

- AI-driven tokenization of real estate, bonds, or commodities
- Intelligent RWA portfolio management agent
- Automated KYC / compliance review tools
- RWA yield aggregator

### **Requirements**

- Project must involve Real World Assets
- Deploy on Mantle Network
- Submit open-source repo + demo + one-line pitch

### **Tell Us in Your Submission**

What type of real-world asset are you bringing on-chain? How does AI play a role? How is it realized on Mantle?

# **Agentic Economy Track _(Exclusively Supported by Byreal)_**

No DeFi expertise required — we provide the full Skills and CLI toolkit. Just bring your creativity.

### **Two Winning Paths**

1. **\[Human Driven\] DeFi Deep Dive** — Use Byreal Agent Skills / Byreal Perps CLI / RealClaw to explore advanced on-chain trading strategies
2. \[AI Driven\] **RealClaw Real-Life Expansion** — Take RealClaw beyond DeFi and into real-world, everyday use cases

### **Core Components**

| Component | Capability | Link |
| :-- | :-- | :-- |
| Byreal Agent Skills | Calls Byreal CLMM, supports LP & Swap | [byreal-agent-skills](https://github.com/byreal-git/byreal-agent-skills) |
| Byreal Perps CLI | Calls Byreal Perps, executes perpetual futures trades | [byreal-perps-cli](https://github.com/byreal-git/byreal-perps-cli) |
| RealClaw | Openclaw-based Agent with Byreal Skills pre-installed, extensible to non-DeFi scenarios | [realclaw](https://www.byreal.io/en/realclaw) |

### **Scoring**

**General (70%):** Byreal integration depth / Agent autonomy / Technical completeness / Sustainability

**Track-Specific (30%)**

- **DeFi Deep Dive → Strategy Alpha:** Strategy complexity + verifiability (backtesting / live trading / on-chain records)
- **RealClaw Expansion → Real-World Validity:** Genuine use of Byreal / RealClaw on-chain capabilities + clear target user in a real-life scenario

### **Encouraged Directions**

**A. DeFi Deep Dive**

- News-driven trading agent using Byreal Agent Skills / Byreal Perps CLI
- Market-making or arbitrage strategies
- Automated portfolio rebalancing with on-chain execution

**B. RealClaw Real-Life Expansion**

- Personal CFO Agent
- On-chain life manager
- Health data management
- Everyday decision-making assistant

### **Requirements**

- Must use core capabilities of at least one of: Byreal Agent Skills / Byreal Perps CLI / RealClaw
- Deploy on Mantle or Solana
- Submit open-source repo + runnable demo + one-line pitch

### **Tell Us in Your Submission**

Which Byreal on-chain capabilities does your project use? What scenario are they applied to?

# **Community Voting**

Decided entirely by the community — recognizing the project with the greatest public appeal and reach.

### **How It Works**

- All submitted projects are automatically eligible
- Voting is open to everyone on the X Platform Voting
- The project with the most votes wins

### **What Wins Here**

- A demo that's clear and compelling — even to non-technical users
- A solution that resonates with real pain points
- Strong community presence and shareability

# **20 Project Deployment Award**

Awarded to the first 20 projects that successfully hit deployment milestones — because shipping matters.

### **How to Qualify**

Meet **all** of the following criteria _(first-come, first-served — 20 spots only)_:

**Technical Deployment**

- ✅ Smart contract deployed on Mantle Mainnet or Testnet
- ✅ Contract is verified on Mantle Explorer
- ✅ At least **one AI-powered function** is callable on-chain (e.g. agent trigger, inference result written on-chain, automated execution)

**📦 Product Completeness**

- ✅ Frontend demo is publicly accessible (not localhost)
- ✅ Deployment address included in your DoraHacks submission
- ✅ Submit a demo video (≥ 2 min) walking through the core use case

**📄 Documentation**

- ✅ Open-source GitHub repo with README (setup instructions, architecture overview, deployed contract address)

### **Note**

This award has no judge scoring — but it has real bars to clear. Every requirement exists to ensure what you ship is genuinely usable, not just a skeleton. Meet all criteria and the award is yours.
